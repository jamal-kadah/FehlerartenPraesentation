Fehlerarten in der Programmierung

Syntaxfehler

Syntaxfehler treten auf, wenn der Programmcode nicht den Regeln der Programmiersprache entspricht.

Beispielcode:

print("Hallo Welt

Fehlermeldung in Visual Studio:

Erklärung: In diesem Beispiel fehlt das schließende Anführungszeichen.

Behebung:

print("Hallo Welt")

Logikfehler

Logikfehler sind Fehler, bei denen der Code zwar ausgeführt wird, aber nicht das gewünschte Ergebnis liefert.

Beispielcode:

zahl = 10
if zahl > 20:
    print("Zahl ist größer als 20")
else:
    print("Zahl ist kleiner oder gleich 20")

Ausgabe in Visual Studio:

Erklärung: In diesem Beispiel ist die Bedingung falsch gesetzt. Die Zahl 10 ist nicht größer als 20.

Behebung:

zahl = 10
if zahl < 20:
    print("Zahl ist kleiner als 20")
else:
    print("Zahl ist größer oder gleich 20")

Laufzeitfehler

Laufzeitfehler treten während der Programmausführung auf und führen oft zu einem Absturz.

Beispielcode:

zahl = 10
ergebnis = zahl / 0

Fehlermeldung in Visual Studio:

Erklärung: In diesem Beispiel führt die Division durch Null zu einem Fehler.

Behebung:

zahl = 10
if zahl != 0:
    ergebnis = zahl / zahl
else:
    ergebnis = 0

Tipps zur Fehlervermeidung

  • Schreibe klaren und gut strukturierten Code.
  • Verwende Kommentare zur Erklärung komplexer Codeabschnitte.
  • Nutze Debugging-Tools, um Fehler frühzeitig zu erkennen.
  • Schreibe Tests, um sicherzustellen, dass dein Code wie erwartet funktioniert.
  • Refaktorisiere regelmäßig deinen Code, um ihn sauber und verständlich zu halten.
