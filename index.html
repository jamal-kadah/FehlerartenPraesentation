Fehlerarten in der Programmierung

Syntaxfehler Logikfehler Laufzeitfehler

Syntaxfehler

Entstehen durch Verstöße gegen die Syntaxregeln einer Programmiersprache. Vom Compiler vor der Ausführung erkannt.

Beispielcode:

Console.WriteLine("Hallo Welt);

Fehlermeldung in Visual Studio:

Syntaxfehler Beispiel

Erklärung:

In diesem Beispiel fehlt das schließende Anführungszeichen.

Behebung:

Console.WriteLine("Hallo Welt");
Zum Anfang

Logikfehler

Das Programm läuft, liefert aber nicht die erwarteten Ergebnisse. Oft schwer zu erkennen und zu beheben.

Beispielcode:

int faktor = 5;
int ergebnis = 1;
for (int i = 1; i <= faktor; i++)
{
    ergebnis += i;
}
Console.WriteLine("Faktor: " + ergebnis);

Fehlermeldung in Visual Studio:

Logikfehler Beispiel

Erklärung:

In diesem Beispiel wird die Addition anstelle der Multiplikation für die Berechnung des Faktors verwendet.

Behebung:

int faktor = 5;
int ergebnis = 1;
for (int i = 1; i <= faktor; i++)
{
    ergebnis *= i;
}
Console.WriteLine("Faktor: " + ergebnis);
Zum Anfang

Laufzeitfehler

Tritt während der Programmausführung auf, oft durch unerwartete Eingaben oder Bedingungen. Kann zum Programmabsturz führen.

Beispielcode:

int zahl = 10;
int ergebnis = zahl / 0;

Fehlermeldung in Visual Studio:

Laufzeitfehler Beispiel

Erklärung:

In diesem Beispiel führt die Division durch Null zu einem Fehler.

Behebung:

int zahl = 10;
if (zahl != 0)
{
    int ergebnis = zahl / zahl;
}
else
{
    int ergebnis = 0;
}
Zum Anfang

Tipps zur Fehlervermeidung

  • Schreibe klaren und gut strukturierten Code.
  • Verwende Kommentare zur Erklärung komplexer Codeabschnitte.
  • Nutze Debugging-Tools, um Fehler frühzeitig zu erkennen.
  • Schreibe Tests, um sicherzustellen, dass dein Code wie erwartet funktioniert.
  • Refaktorisiere regelmäßig deinen Code, um ihn sauber und verständlich zu halten.
